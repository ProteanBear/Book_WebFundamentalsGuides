# Web基础指南

## 介绍

分享来自Google的《Web基础指南》，即此[网站](https://developers.google.com/web/fundamentals/)的内容。内容中会分享一些来自谷歌创建Web站点的参考意见。  
此版本为中文版，用于帮助中国境内可能因为某些原因无法访问此网站内容的人。因为书中本身的内容就已经翻译成了中文所以此部分只是分享，而有些不是中文的部分本人会翻译为中文，方便阅读。
